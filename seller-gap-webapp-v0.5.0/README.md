# 셀러 갭 소싱 웹앱 v0.5.0

카테고리에서 후보 키워드를 생성한 뒤 네이버와 쿠팡을 분리해 조사하는 Windows용 로컬 웹앱입니다.

## 이번 버전 핵심

- 1차, 2차, 3차, 4차 어느 단계에서든 카테고리 조사 가능
- 선택 단계 아래의 하위 분류와 편집 가능한 키워드 사전에서 후보 자동 생성
- 네이버 상품 / 쿠팡 / 통합 비교 탭
- 쿠팡 평균가, 최저가, 중앙가, 총 리뷰, 최대 리뷰, 평균 리뷰
- 쿠팡 로켓배송, 판매자로켓, 일반배송, 해외배송, 광고상품 비율
- 모든 쿠팡 수치에 실제 수집 표본 수와 수집 출처 표시
- 결과 키워드를 클릭하면 상세 분석 화면 표시
- 네이버 검색 추이와 쇼핑 클릭 추이 그래프
- 네이버·쿠팡 상품 요약과 쿠팡 상위 상품 목록
- 연관 키워드와 상품명 빈도 분석
- 쿠팡 수집 실패 시 저장한 검색 페이지 HTML을 붙여 넣어 재계산

## 설치와 실행

1. ZIP 압축을 새 폴더에 풉니다.
2. `seller-gap-webapp-v0.5.0` 폴더로 들어갑니다.
3. 최초 1회 `install.bat`을 실행합니다.
4. 쿠팡 로그인 표본을 사용할 경우 최초 1회 `coupang_login.bat`을 실행해 별도 Chrome 창에서 로그인합니다.
5. `run.bat`을 실행합니다.
6. 브라우저가 열리지 않으면 `http://127.0.0.1:8000`에 접속합니다.

`install.bat`은 Chrome 검색결과를 읽기 위한 Python Playwright 패키지를 설치합니다. 별도의 Chromium을 내려받지 않고 PC에 설치된 Chrome을 우선 사용합니다.

문법과 기본 계산을 확인하려면 `check.bat`을 실행합니다. `Validation passed.`가 나오면 Python 파일과 기본 쿠팡 통계 계산기가 정상입니다.

## API 설정

### NAVER API HUB — 필수

Application에 아래 두 기능을 등록합니다.

- 검색어 트렌드
- 쇼핑 인사이트

웹앱 우측 상단 `API·수집 설정`에 Client ID와 Client Secret을 저장합니다.

### 네이버 검색광고 API — 선택

절대 월 검색량과 연관 키워드가 필요할 때 설정합니다.

- API Key
- Secret Key
- Customer ID

설정하지 않아도 NAVER API HUB 상대 추이와 상품 검색 표본은 사용할 수 있습니다. 설정하지 않은 경우 월 검색량은 `미설정`으로 표시됩니다.

## 쿠팡 수집 방식

### Chrome 로그인 세션 사용

권장 방식입니다.

- `coupang_login.bat`을 실행하고 열린 Chrome에서 쿠팡에 로그인합니다.
- 로그인이 끝나면 검은 창으로 돌아가 Enter를 누릅니다.
- 로그인 상태는 웹앱 폴더의 `data/browser-profile`에 저장됩니다.
- 조사할 때 같은 별도 프로필의 Chrome 창이 열리고 검색결과를 읽은 뒤 자동으로 닫힙니다.

### 서버 직접 요청

Chrome을 열지 않지만 쿠팡이 요청을 제한하면 상품을 읽지 못할 수 있습니다.

### 저장 HTML 분석

키워드 상세 분석 화면에서 `저장 HTML로 다시 계산`을 누르면 쿠팡 검색 페이지의 HTML을 붙여 넣어 가격, 리뷰, 배송 비율을 다시 계산할 수 있습니다.

## 데이터 해석

### 네이버 검색 추이

네이버 통합검색 상대지수입니다. 조회 기간 중 가장 큰 값을 100으로 둡니다.

### 네이버 쇼핑 클릭 추이

네이버 통합검색 쇼핑 영역과 네이버쇼핑의 상대 클릭 추이입니다. 쿠팡 데이터가 아닙니다.

### 네이버 월 검색량

네이버 검색광고 API를 연결했을 때만 표시하는 PC·모바일 검색량 합계입니다.

### 쿠팡 통계

쿠팡 검색결과에서 실제로 읽힌 상품 표본으로 계산합니다.

- 평균가 = 가격을 읽은 상품 가격 합계 / 가격 표본 수
- 총 리뷰 = 리뷰 수를 읽은 상품의 리뷰 합계
- 최대 리뷰 = 리뷰 표본 중 최댓값
- 평균 리뷰 = 리뷰 합계 / 리뷰 수를 읽은 상품 수
- 배송 비율 = 해당 배송 유형 상품 수 / 전체 수집 상품 수

쿠팡 전체 월 검색량, 전체 클릭량, 경쟁 판매자의 실제 판매량은 공개값이 아니므로 만들지 않습니다.

## 카테고리 편집

`data/categories.json`을 메모장으로 수정하면 카테고리와 자동 생성 키워드를 추가할 수 있습니다. JSON 문법이 깨지면 카테고리가 로드되지 않으므로 수정 전 사본을 보관하세요.

## 보안

- API 비밀키는 `data/config.json`에 평문으로 저장됩니다.
- 쿠팡 로그인 쿠키는 `data/browser-profile`에 저장됩니다.
- 폴더를 다른 사람에게 전달하기 전에 `data/config.json`과 `data/browser-profile`을 삭제하세요.
- 공유 PC에서는 사용하지 않는 편이 안전합니다.

## 제한

- 쇼핑 사이트의 HTML 구조가 바뀌면 상품 수집 규칙을 수정해야 할 수 있습니다.
- 쿠팡 로그인, CAPTCHA 또는 접근 제한이 발생하면 자동 수집이 실패할 수 있습니다.
- 수집 표본은 검색순위, 로그인 상태, 지역, 시각에 따라 달라질 수 있습니다.
- 숫자가 존재하지 않을 때 다른 플랫폼 값을 복사하거나 임의 추정하지 않습니다.
